## 🚀 Getting Started

***Requirements***

Ensure you have the following dependencies installed on your system:

* **JavaScript**: `version x.y.z`

### ⚙️ Installation

1. Clone the octokit-node-crud repository:

```sh
git clone https://github.com/SpectralAUT/octokit-node-crud
```

2. Change to the project directory:

```sh
cd octokit-node-crud
```

3. Install the dependencies:

```sh
npm install
```

### 🤖 Running octokit-node-crud

Use the following command to run octokit-node-crud:

```sh
node app.js
```

### 🧪 Tests

To execute tests, run:

```sh
npm test
```
